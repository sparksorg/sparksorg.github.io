Object-Oriented Programming in Java

What is this course?

This course is an introduction to object oriented programming in Java. This course offers a comprehensive introduction to object oriented programming in Java. By the end of the course, students will have the ability to create a variety of projects in Java including a basic video game using JAVA.In addition, students will gain a solid foundation of object oriented programming and the basic theory behind it.

Why take this course?

Java is one of the most widely used programming languages for Web applications and platforms. Many jobs in the programming industry require knowledge of Java and extensively use it for their products.

Students will use the skills in taught in this Java course in many of their college/university courses and in many of their future jobs in the programming industry. Over 3 billion devices run Java and many major companies, like Google, Uber, and Netflix utilize Java in their tech stacks. It is used everywhere from server side applications to high frequency trade applications.

Web Development using HTML, CSS, and JS

What is this course?

This is a course on web development which offers a comprehensive introduction to the huge world of website development. By the end of the course, students will have the knowledge and ability to create their own professional-looking website—all from scratch.

Why take this course?

You’ve probably used an internet browser before. Google Chrome, Firefox, Internet Explorer, Safari, Opera, or even Tor are all popular. You probably already know that these browsers are the tools that open up websites.

But understanding how these browsers work is essential to learning website development. When you navigate to a website (for example, sparksorg.github.io.com), you actually download the html file associated with the webpage. Don’t worry - it’s only temporary! The browser downloads the files, and takes those files and interprets them.

What does that mean? Websites are written in HTML, a markup language. The browser goes through this markup language, identifying each and every tag, and displaying the content the website creator wants it to onto the screen. This is called the Hypertext Transfer Protocol (HTTP). Have you heard of this acronym, or even HTTPS before?

Well, you will learn about this and much more in this course about web development, an essential skill for every student to have.
